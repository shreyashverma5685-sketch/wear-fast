# WEAR FAST #

A smart wardrobe management and outfit recommendation platform.

##  Vision
Help users manage their wardrobe digitally, get AI-powered outfit suggestions based on weather, occasion, and personal style — and never stare at a full closet wondering what to wear.

##  Tech Stack
| Layer | Tech |
|-------|------|
| Frontend | HTML/CSS/JS → React |
| Backend | Node.js + Express |
| Database | MongoDB |
| Auth | JWT + bcrypt |
| Images | Cloudinary |
| Weather | OpenWeatherMap API |
